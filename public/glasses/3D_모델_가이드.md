# 3D 안경 오버레이용 모델 (.glb)

3D 오버레이를 쓰려면 **.glb** (또는 .gltf) 형식의 안경 3D 모델이 필요합니다.  
`public/glasses/` 폴더에 `.glb` 파일을 넣고, 상품에 `modelUrl`을 지정하면 됩니다.

## 무료 3D 안경 모델 다운로드

| 출처 | 설명 | 형식 | 비고              |
|------|------|------|-----------------|
| [Get3DModels – Glasses Low Poly](https://www.get3dmodels.com/fashion/glasses-low-poly-6/) | 저폴리 안경 (~39KB) | GLB | 가벼워 웹에 적합       |
| [Get3DModels – Glasses](https://www.get3dmodels.com/tools-and-gadgets/glasses/) | 안경 3D 모델 | GLB |                 |
| [CGTrader – 3D Sun Glasses](https://www.cgtrader.com/free-3d-models/character/clothing/3d-sun-glasses) | 선글라스, low-poly | OBJ, GLB, FBX | 무료 계정 후 다운로드    |
| [Sketchfab – Glasses](https://sketchfab.com/3d-models/glasses-4b8a1e54f3084c63828fe8c324198aec) | 다양한 안경 모델 | GLB 등 | 일부 무료, 라이선스 확인  |
| [glTF Viewer](https://gltf-viewer.donmccurdy.com/) | 다양한 안경 모델 | GLB 등 | 진짜 무료, 가입 필요 없음 |

- **OBJ**만 있는 경우: [Blender](https://www.blender.org/) 등으로 불러온 뒤 **File → Export → glTF 2.0 (.glb)** 로 변환하면 됩니다.
- MediaPipe Face Landmarker의 **facial transformation matrix**와 함께 쓰면, 얼굴 포즈에 맞춰 3D로 붙습니다. 모델은 “정면 기준”이면 되고, 크기/위치는 코드에서 스케일·오프셋으로 조정할 수 있습니다.

## 이 프로젝트에서 사용하는 방법

1. **.glb 파일 배치**  
   예: `public/glasses/glasses.glb` 또는 `public/glasses/sunglasses.glb`

2. **상품 데이터에 modelUrl 지정**  
   `src/data/sampleGlasses.ts`에서 해당 상품에 `modelUrl: '/glasses/glasses.glb'` 처럼 추가합니다.

3. **실행**  
   `npm run dev` 후 해당 안경을 선택하면 3D 모델이 얼굴에 오버레이됩니다.

- `modelUrl`이 **없으면** → 2D 안경 이미지(`imageUrl`)를 3D 평면 텍스처로 씁니다.  
- `modelUrl`이 **있으면** → 해당 .glb를 로드해 3D 메쉬로 렌더링합니다.

## 참고: MediaPipe 3D Face

- [MediaPipe 3D Face Transform](https://developers.googleblog.com/en/mediapipe-3d-face-transform)  
  Canonical face model + transformation matrix로 3D 오브젝트를 얼굴에 붙이는 방식입니다.  
  이 프로젝트도 같은 matrix를 사용하므로, “얼굴에 붙는” 3D 안경 모델이면 그대로 활용 가능합니다.
