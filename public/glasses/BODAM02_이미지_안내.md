# BODAM 02 (다비치마켓) 3D 뷰어용 이미지

- 상품 페이지: https://www.davichmarket.com/products/G2201191207_8901

## 필요한 파일

1. **bodam02-front.png** – 안경 **정면** 사진 (앞면)
2. **bodam02-side.png** – 안경 **측면** 사진 (옆면, 좌/우 동일 적용)

이 두 파일을 이 폴더(`public/glasses/`)에 넣으면 3D 안경 뷰어(`/viewer`)에서 BODAM 02가 앞·옆 텍스처로 표시됩니다.

## 이미지 얻는 방법

1. 위 상품 페이지를 브라우저에서 연다.
2. 상품 이미지 갤러리에서 **정면**에 해당하는 이미지를 저장 → `bodam02-front.png`로 저장.
3. **측면**에 해당하는 이미지를 저장 → `bodam02-side.png`로 저장.
4. 두 파일을 `public/glasses/`에 넣고 앱을 다시 실행한 뒤 `/viewer`에서 확인.

이미지를 넣기 전에는 뷰어가 기본 이미지(`glasses.png`)로 동작합니다.
